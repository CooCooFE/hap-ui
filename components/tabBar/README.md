# ux-tabBar 

> quickApp 版本底部 tab-bar 组件


### 可配置参数

| Prop | Type | Required | Default | Description |
|-------------|------------|--------|-----|-----|
| mode | `String` |`N`| `fixed` | 是否开启tab-bar横向滚动 |
| index | `Number` |`N`| `0` | 当前active tab索引 |
| tabtitles | `Array` |`Y`| `[]` | tab-bar显示 配置 |
